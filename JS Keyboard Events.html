Keyboard Events များအကြောင်း

နည်းပညာရှုထောင့်မှ တိုက်ရိုက်နားလည်စေရန်နှင့် လက်တွေ့လုပ်ငန်းခွင်တွင် အသုံးပြုပုံများကို အသေးစိတ် ရှင်းလင်းတင်ပြထားပါသည်။

Technical Explanation

Web Browser ပေါ်တွင် အလုပ်လုပ်နေစဉ် ကီးဘုတ် (Keyboard) ပေါ်မှ ခလုတ်တစ်ခုခုကို နှိပ်လိုက်သည့်အခါ၊ ဖိထားသည့်အခါ၊ သို့မဟုတ် ပြန်လွှတ်လိုက်သည့်အခါ Browser မှ သိရှိပြီး ထုတ်ပေးလိုက်သော Signal (အချက်ပြမှု) များကို Keyboard Events ဟု ခေါ်ပါသည်။

ဤ Events များသည် Document Object Model (DOM) မှတစ်ဆင့် JavaScript သို့ သတင်းအချက်အလက်များ ပို့ဆောင်ပေးပါသည်။

Equation Representation:

$$ \text{Event}_{\text{total}} = \text{keydown} + \text{keypress} + \text{keyup} $$
Simply Put

ရိုးရှင်းသော ရှင်းလင်းချက်

ဒါဘာလဲ?

ကီးဘုတ်အသုံးပြုမှုကို စောင့်ကြည့်နားထောင်ပေးသော စနစ်ဖြစ်ပြီး၊ အသုံးပြုသူက ကီးဘုတ်မှတစ်ဆင့် Data ထည့်သွင်းခြင်း၊ ဂိမ်းကစားခြင်း သို့မဟုတ် Shortcut များကို အသုံးပြုလိုသည့်အခါ JavaScript ဖြင့် တုံ့ပြန်မှုများ ဖန်တီးရန်ဖြစ်သည်။

ဘယ်အချိန်သုံးသလဲ?

အသုံးပြုသူက ကီးဘုတ်ခလုတ်ကို စတင်ထိလိုက်သည် (Press)၊ ဖိထားသည် (Hold)၊ သို့မဟုတ် ပြန်လွှတ်လိုက်သည် (Release) ဆိုသည့် အချိန်ကာလအပိုင်းအခြားများတွင် အသုံးပြုပါသည်။

Analogy

လူအိမ်ရှေ့က တံခါးဘဲလ် (Doorbell)

လက်က ဘဲလ်ခလုတ်ကို စတင်ထိလိုက်ပြီး ဖိလိုက်တဲ့အချိန်
event: keydown
လက်က ဘဲလ်ခလုတ်ကို မလွှတ်ဘဲ ဆက်တိုက် ဖိထားခြင်း
event: keypress (repeating)
လက်ကို ခလုတ်ပေါ်မှ ဖယ်လိုက်ပြီး ခလုတ်ပြန်ကြွလာတဲ့အချိန်
event: keyup

အဓိက မှတ်ရမယ့် အချက်များ

Events (၃) ခု

keydown (ခလုတ်နှိပ်စ)၊ keypress (စာလုံးရိုက်ခြင်း - deprecated)၊ နှင့် keyup (ခလုတ်လွှတ်ခြင်း)။

event.key vs event.code

event.key: ရိုက်လိုက်သော တန်ဖိုး (ဥပမာ - "a", "A", "Enter")။

event.code: ကီးဘုတ်ပေါ်ရှိ ခလုတ်နေရာ (ဥပမာ - "KeyA")။

Focus & Default

Event အလုပ်လုပ်ရန် Element သည် Focus ရရှိနေရန် လိုသည်။ Browser မူလလုပ်ဆောင်ချက် (ဥပမာ F5) ပိတ်ရန် event.preventDefault() သုံးရသည်။

လက်တွေ့ အသုံးပြုပုံ (Real World)

(က) keydown Event

ဂိမ်းများတွင် ဇာတ်ကောင် ရွေ့လျားရန်၊ Shortcut Key များ (Ctrl+C, Ctrl+V) ဖမ်းယူရန် သုံးသည်။

// အသုံးပြုသူက Enter ခလုတ်နှိပ်လျှင် Login ဝင်မည့် Function
document.addEventListener('keydown', function(event) {
    // event.key သည် နှိပ်လိုက်သော တန်ဖိုးကို စစ်ဆေးခြင်း
    if (event.key === 'Enter') {
        console.log('Login လုပ်ဆောင်ချက် စတင်ပါပြီ...');
        // performLogin();
    }
});

(ခ) keyup Event

စာရိုက်ပြီးသွားမှ အလုပ်လုပ်စေချင်သည့်အခါ (ဥပမာ - Search Box တွင် စာရိုက်ပြီးမှ ရှာဖွေခြင်း) သုံးသည်။

const searchInput = document.getElementById('search-box');

searchInput.addEventListener('keyup', function(event) {
    // လက်ရှိ ရိုက်ထားသော စာသားကို ယူမည်
    const query = event.target.value;
    
    // စာလုံး လွှတ်လိုက်တိုင်း Console တွင် ပေါ်မည်
    console.log("ရှာဖွေနေသည်: " + query);
});

သဘောတရား ပုံကြမ်း

လက်စတင်ထိ
keydown
လက်ဖိထားဆဲ
repeating
လက်ပြန်ကြွ
keyup
Real World Scenario (ဂိမ်း):
  • keydown: ကား စတင် ကွေ့သည်။ (တာယာဘီး လှည့်သွားသည်)
  • Hold Key: ကား ဆက်လက် ကွေ့နေသည်။
  • keyup: ကား ကွေ့ခြင်း ရပ်သွားသည်။ (တာယာဘီး တည့်သွားသည်)
Example Project

Modal Box ပိတ်ခြင်း

Modal Box (Pop-up) တစ်ခု ပွင့်နေချိန်တွင် Escape (Esc) ခလုတ် နှိပ်လိုက်ပါက Modal ကို ပိတ်ပေးရမည့် လုပ်ဆောင်ချက်။

// Modal Element ကို လှမ်းယူခြင်း
const modal = document.querySelector('.modal');

// Document တစ်ခုလုံးကို နားထောင်မည်
document.addEventListener('keydown', function(event) {
    
    // နှိပ်လိုက်သော ခလုတ်သည် Escape ဖြစ်ပြီး Modal ဖွင့်နေလျှင်
    if (event.key === 'Escape' && !modal.classList.contains('hidden')) {
        
        // Modal ကို ပိတ်မည့် Function ခေါ်ခြင်း
        closeModal();
        console.log('Escape key ဖြင့် Modal ကို ပိတ်လိုက်ပါပြီ။');
    }
});

function closeModal() {
    modal.classList.add('hidden');
}
